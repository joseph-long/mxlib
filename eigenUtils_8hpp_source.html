322  * \param [out] eigval will contain the eigenvalues

324  * \param [in] X is a square matrix which is either upper or lower (default) triangular
325  * \param [in] ev0 is the first desired eigenvalue default 0
326  * \param [in] ev1 if >= ev0 thenthis is the last desired eigenvalue. If -1 all eigenvalues are returned.
327  * \param [in] UPLO specifies whether X is upper ('U') or lower ('L') triangular. Default is ('L').
