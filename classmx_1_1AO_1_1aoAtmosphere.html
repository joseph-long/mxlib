◆ dir_wind()

template<typename realT >
realT mx::AO::aoAtmosphere< realT >::dir_wind ( )

Get the weighted mean wind direction.

Returns the weighted mean wind direction according to the 5/3's turbulence moment. This is defined as

\[ \bar{\theta} = \left[\sum_i C_N^2(z_i) \theta_i^{5/3} \right]^{3/5} \]

See Hardy (1998) Section 3.3.6. [9].

This is only re-calculated if either _layer_Cn2 or _layer_v_wind is changed, otherwise this just returns the value of _dir_wind.

Returns
the current value of _dir_wind.

◆ fwhm()

template<typename realT >
realT mx::AO::aoAtmosphere< realT >::fwhm ( realT  lam_sci)

Calculate the full-width at half-maximum of a seeing limited image for this atmosphere.

Calculate the FWHM of a seeing limited image with the current parameters according to Floyd et al. (2010) [4]

\[ \epsilon_0 = 0.98\frac{\lambda_{sci}}{r_0(\lambda_sci)}. \]

If there is an outer scale (_L_0 > 0), then a correction is applied according to Tokovinin (2002) [22]

\[ \left( \frac{\epsilon_{vK}}{\epsilon_0}\right)^2 = 1 - 2.183\left( \frac{r_0(\lambda_{sci}}{L_0}\right)^{0.356} \]

Returns
the value of the FWHM ( \( \epsilon_{0/vK} \)) for the current atmosphere parameters.
Parameters
[in]lam_scithe wavelength of the science observation.
