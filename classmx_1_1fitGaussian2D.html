template<typename fitterT>
class mx::fitGaussian2D< fitterT >

Class to manage fitting a 2D Gaussian to data via the levmarInterface.

In addition to the requirements on fitterT specified by levmarInterface this class also requires this definition in fitterT

static const int nparams = 7;

where the number 7 is replaced by the number of parameters that fitterT expects to fit.
