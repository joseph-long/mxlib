sharedmem_segment Struct Reference
General Utilities » System Programming » Interprocess Communication » Shared Memory

A c structure to manage a shared memory segment with memory mapping.

The associated functions create and/or attach to a shared memory segment. If a segment is created, a block of size sizeof(uintptr_t) is reserved where the address is stored. This address is used when subsequently attaching to the segment so that pointers stored in the block are valid, etc.

See also
Functions for working with sharedmem_segment include: sharedmem_segment_initialize, sharedmem_segment_set_key, sharedmem_segment_create, sharedmem_segment_attach, and sharedmem_segment_detach
