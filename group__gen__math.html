Calculate the pseudo-inverse of a patrix using the SVD.

First computes the SVD of A, \( A = U S V^T \), using eigenGESDD. Then the psuedo-inverse is calculated as \( A^+ = V S^+ U^T\). This interface does not provide access to U, S and VT.

condition[out] The final condition number.
nRejected[out] the number of eigenvectors rejected
A[in] the matrix to invert

interact[in] [optional] a bitmask controlling interaction. If (interact & MX_PINV_PLOT) is true, then gnuPlot is used to display the eigenvalues. If (interact & MX_PINV_ASK) is true then the minimum eigenvaue threshold is requested from the user using stdin.

Find the roots of the general quartic equation.

Finds the roots of

\[ f(x) = a x^4 + b x^3 + c x^2 + d x + e \]

using the general formula for quartic roots. See https://en.wikipedia.org/wiki/Quartic_function.

Parameters
[out]xis resized to length 4, and on exit contains the 4 roots.
[in]ais the coefficient of the \(x^4\) term.
[in]bis the coefficient of the \(x^3\) term.
[in]cis the coefficient of the \(x^2\) term.
[in]dis the coefficient of the \(x^1\) term.
[in]eis the coefficient of the \(x^0\) term.
Template Parameters
realTis the floating point type used for calculations.
