86 /**
87  * @brief The Laplace (double exponential) continuous distribution for random numbers.
88  *
89  * The formula for the exponential probability density function is
90  * @f$p(x|\lambda) = \frac{\lambda}{2} e^{-\lambda x}@f$.
91  *
92  * <table border=1 cellpadding=10 cellspacing=0>
93  * <caption align=top>Distribution Statistics</caption>
94  * <tr><td>Mean</td><td>@f$zero@f$</td></tr>
95  * <tr><td>Median</td><td>@f$zero@f$</td></tr>
96  * <tr><td>Mode</td><td>@f$zero@f$</td></tr>
97  * <tr><td>Range</td><td>@f$[-\infty, \infty]@f$</td></tr>
98  * <tr><td>Standard Deviation</td><td>@f$\frac{\sqrt{2}}{\lambda}@f$</td></tr>
99  * </table>
100  *
101  * This is based on the implementation of the exponential distribution in the GNU ISO C++ Library version 4.6.

103  * \ingroup random
