template<typename T >
T mx::jinc ( const T &  x)

Returns the the Jinc function.

The Jinc function is defined here as

\[ Ji(x) = \frac{J_1(x)}{x} \]

where \( J_1 \) is the cylindrical bessel function of the first kind of order 1.

Follows the technique in boost sinc_pi, using the Taylor series for small arguments. If x is smaller than \( \epsilon \), then it returns 1/2. If x is larger than \( \epsilon \) but smaler than \( \sqrt{\epsilon} \), then this function returns

\[ Ji(x) \approx \frac{1}{2} - \frac{x^2}{16}. \]

Returns the Jinc2 function.

The Jinc2 function is defined here as

\[ Ji_2(x) = \frac{J_2(x)}{x} \]

where \( J_2 \) is the cylindrical bessel function of the first kind of order 2.

If x is smaller than \( \sqrt{\epsilon} \), returns 0.

Return the logistic function parameter for a specified rise time.

The logistic function is

\[ f(t) = \frac{1}{1 + e^{-a(t-t_0)}} \]

The parameter \( a \) controls how fast the function rises. Here it is specified by the value \( f(t_{1/2}) = x\), where \( 0 < x < 1 \).

x[input] the value at which the rist time is specified.
thalf[input] half the rise time, or the time after 0 when f(t) = x.
