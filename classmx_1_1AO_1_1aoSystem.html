template<typename realT, class inputSpectT, class wfsBetaT>
class mx::AO::aoSystem< realT, inputSpectT, wfsBetaT >

Describes an analytic adaptive optics (AO) system.

Templatized by the turbulence power spectral density (PSD) and the wavefront sensor (WFS) sensitivity function.

Template Parameters
realTthe floating point type used for all calculations
inputSpecTspecifies the turbulence spatial PSD type
wfsBetaTspecifies the WFS sensitivity type.
