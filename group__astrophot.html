Read a filter transmission curve from the astrofilt collection.

Reads in a filter transmission curve from the astrofilt collection, usually found via the ASTROFILT_DATADIR environment variable. Astrofilt profiles are in wavelength of microns, and transmission normalized to peak of 1 and airmass 0.

Parameters
lam[out] the profile wavelength scale, in microns. Is cleared.
trans[out] the transmission at each lambda, normalized to a peak of 1 at 0 airmass. Is cleared.
filtName[in] the name of the filter, with no path or extension, e.g. 'J_NIRC2' or 'ip_VisAO'
rsr[in] [optional] if true converts to relative spectral response, appropriate for photon detectors (like CCDs). See Bessell, PASP, 112, 961 (2000).
datadir[in] [optional] specifies the data directory. if not set or empty, the environment is queried for then value of ASTROFILT_DATADIR.
Template Parameters
dataTthe type of data to return

Characterize the wavelength of a filter transmission curve.

For a transmission curve given by \( T(\lambda ) \) The central wavelength is defined as \( \lambda_0 = \frac{1}{w_{eff}}\int T(\lambda ) \lambda d\lambda \) where the effective width is defined by \( w_{eff} = \int T(\lambda ) d\lambda \)

The full-width at half-maximum, FWHM, is the distance between the points at 50% of maximum \( T(\lambda) \).

Parameters
lambda0[out] the central wavelength of the filter
fwhm[out] the full-width at half-maximum of the filter profile
weff[out] the effective width of the filter
lambda[in] the wavelength grid of the profile
trans[in] the transmission of the profile at each wavelength
Template Parameters
dataTthe data type in which calculations are performed.

Convert a transmission profile to relative spectral response.

The relative spectral response is given by

\( T_{RSR}(\lambda) = T(\lambda)\lambda / \mbox{max}[T(\lambda)\lambda] \)

where \( T(\lambda) \) is the energy transmission profile. This is appropriate for photon detectors (like CCDs). See Bessell, PASP, 112, 961 (2000).

Parameters
lambda[in] The wavelength scale
trans[in] The transmission profile
